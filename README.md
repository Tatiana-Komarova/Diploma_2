### API тесты для проверки сайта StellarBurgers

### Реализованные сценарии

Созданы тесты тестирующие API эндпоинты: `Создание пользователя`, `Логин пользователя`, `Изменение данных пользователя`, `Создание заказа`, `Получение заказов конкретного пользователя`   


### Структура проекта

- `methods` - пакет, содержащий методы
- `tests` - пакет, содержащий API тесты, разделенные по классам

### <h>1. Установите зависимости:</h>

> pip install -r requirements.txt</h>

### <h>2. Запустить все тесты:</h>

> pytest -v

### <h>3. Посмотреть отчет по прогону html</h>

> allure serve allure-results
